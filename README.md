# NOTAS: 
Aplicación de multiplicaciones con YARGS
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Numero de la base                          [número] [requerido]
  -l, --listar   Listar los numeros a multiplicar
                                         [booleano] [requerido] [defecto: false]
  -h, --hasta    Numero hasta el cual se multiplica       [número] [defecto: 10]
```
